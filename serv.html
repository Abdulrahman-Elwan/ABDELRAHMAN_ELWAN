Experience

In 2019, I began my journey into web development with a focused effort on building my foundational skills and expanding my knowledge in this exciting field. My plan involved learning the key technologies that form the basis of creating websites and interactive web applications.

I started with HTML5, the core language for structuring web pages. This phase took me a week and a half. During this time, I learned how to create the basic structures of pages, insert text, images, and links, and understood the importance of semantic elements for SEO enhancement.
After establishing a solid foundation in HTML5, I moved on to learning CSS3 over a period of 15 days. I learned how to style and design web pages to enhance their visual appeal. The lessons included grid layouts, aesthetic enhancements, and setting up responsive designs that adapt to various screen sizes and devices.

Next, I dedicated two and a half months to learning JavaScript, the fundamental language for adding interactivity to websites. During this period, I learned how to manipulate DOM elements, add dynamic interactions, and handle events. This phase was crucial in enhancing my understanding of how to make web pages more interactive and dynamic.

At the end of my basic language learning, I began studying React.js over a span of four months. React.js is a JavaScript library used for building interactive, component-based user interfaces. I learned how to create reusable components, manage state, and use routing to create Single Page Applications (SPA) with a smooth user experience.

Finally, I turned to learn Next.js over three months, a framework built on React.js that enables advanced web application development. I focused on performance optimization and user experience through Server-Side Rendering (SSR) and Static Site Generation (SSG). This technology enabled me to build fast and secure applications with enhanced SEO performance.

This educational journey enabled me to build a strong foundation in web development and smoothly transition from learning the basics to mastering more advanced tools and technologies. These acquired skills have strengthened my capabilities, making me well-prepared to develop innovative and efficient web projects that meet the needs of clients and the market.
